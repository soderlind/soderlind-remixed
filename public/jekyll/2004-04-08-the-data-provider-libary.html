The .Data Provider Libary is a 100% managed, high-performance library of data providers for use in .NET environments. The library is written entirely in C# and currently supports MySQL versions 3.23x and 4.0x. Support for PostgreSQL and SAPdb are in the works. And best of all, it's free!!
