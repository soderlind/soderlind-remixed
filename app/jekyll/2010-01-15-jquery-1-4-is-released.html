*DRUM ROLL*  jQuery 1.4 is released !!!

It has a lot of new features and enhancements like the examples below:

[sourcecode language="javascript" gutter="true"]
jQuery("<div>", {
"class": "test",
text: "Click me!",
click: function(){
$(this).toggleClass("test");
}
}).appendTo("body");

jQuery("<input>", {
type: "text",
val: "Test",
focusin: function() {
$(this).addClass("active");
},
focusout: function() {
$(this).removeClass("active");
}
}).appendTo("form");
[/sourcecode]
